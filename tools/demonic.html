Zalgo vs. Demonic Text

Many people wonder if Zalgo text and Demonic text are different fonts. The truth is, they both use the same Unicode technology. Zalgo is the original name used by the internet to describe text that appears to be "melting" or "glitched." It uses overlapping symbols to break out of the standard text line. Demonic Text (also called Cursed Text) is simply the modern name for this effect. It is popular because the vertical "dripping" symbols look like blood or static, giving your messages a scary, possessed appearance.
