Is This Asian Style Text Fonts?

Instead of using an actual font file, the generator selects characters from the international Japanese Unicode standard. For example, it may use characters like 乇乂丅尺卂 to represent the word “Extra” based purely on visual shape.

These are decorative symbol writings created for fun and visual style, not for writing real Japanese language content.
